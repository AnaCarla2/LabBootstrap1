Introducción a la Programación con JavaScript

Este curso te enseña los conceptos esenciales de la programación como variables, estructuras de control, funciones y objetos usando JavaScript. Ideal para dar tus primeros pasos en el desarrollo web o cualquier área tecnológica. Duración estimada: 4 semanas (12-16 horas) Nivel: Básico

Desarrollo Web con HTML y CSS

Aprende a crear páginas web desde cero usando HTML5 para estructurar contenido y CSS3 para darle estilo. Este curso combina teoría con práctica para que construyas tus primeras páginas y entiendas cómo funciona la web por dentro. Duración estimada: 3 semanas (10-12 horas) Nivel: Básico

Fundamentos de Java y Programación Orientada a Objetos

Aprende a programar con Java, uno de los lenguajes más demandados por empresas. Conocerás conceptos clave como clases, objetos, herencia y más, aplicando buenas prácticas de desarrollo desde el inicio. Duración estimada: 5 semanas (16-20 horas) Nivel: Intermedio.

Desarrollo Frontend con React.js

Este curso te lleva paso a paso para dominar React, la biblioteca de JavaScript más popular. Construirás componentes reutilizables, manejarás estados y trabajarás con hooks para crear aplicaciones web dinámicas. Duración estimada: 6 semanas (20-25 horas) Nivel: Intermedio.

Bases de Datos con MySQL y SQL

Aprende a trabajar con bases de datos relacionales utilizando MySQL. Crearás esquemas, ejecutarás consultas complejas y comprenderás cómo se integran las bases de datos en aplicaciones reales. Duración estimada: 4 semanas (12-15 horas) Nivel: Intermedio

Desarrollo Backend con Spring Boot

Descubre cómo crear aplicaciones backend usando el framework Spring Boot. Aprenderás sobre controladores, servicios, repositorios, seguridad y conexión con bases de datos, desarrollando APIs listas para producción. Duración estimada: 6 semanas (22-28 horas) Nivel: Avanzado
